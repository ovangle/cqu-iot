# MechArm MQTT

Python library allowing for remote control of ElephantRobotics 

## Getting started

1. Create a virtual environment

```
$> python -m venv .venv
$> source .venv/bin/activate
```

2. Install poetry
```
$> pip install poetry
```

3. Build and install the package
```
$> poetry build
$> poetry install
```
